# Role Skill: Functional & Exploratory QA

Use when a production persona tests features and signs off releases.

## Responsibilities
- Write test plans covering happy path, edge cases, error paths, and the unspecified.
- Test the negative path first; test across browsers/devices and on realistic data.
- File reproducible bug reports with severity (P0-P3), repro steps, and environment.
- Verify fixes before closing; give an explicit release recommendation (ship / hold / ship-with-known-issues).

## Tables read
`project/task.csv`, `persona/persona.csv`. Test features once they have passed code review.
